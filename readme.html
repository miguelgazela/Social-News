Notas adicionais:

  • A página socialnews.php é a página inicial do site. Apresenta as 10 notícias mais recentes da base de dados, ou as que existem caso sejam menos do que 10. Cada notícia possui 3 links: no título da notícia e no 'see more' que levam para a página da notícia e um em 'comments' que leva para a zona dos comentários da página da notícia.
  • A página allnews.php (link News do header) apresenta todas as notícias da base de dados. Carrega um certo número de notícias e à medida que o utilizador vai descendo na página utiliza AJAX e Javascript para ir adicionando outras notícias ao DOM.
  • A página newsSearch.php (link News Search do header) permite fazer pesquisas utilizando os 3 critérios necessários para o protocolo de pesquisa. Utilizando a pesquisa no header (apenas por tags) os resultados são apresentados nesta página.
  • A página newAccount.php (link Create account do header) permite o registo de novo utilizador. Possui verificação por Javascript dos campos do formulário. Esta verificação é feita também no lado do servido caso o utilizador tenha o Javascript desactivado.
  • É possível marcar uma notícia como favorita através de um botão com a imagem de uma estrela presente na notícia.
  • A edição dos comentários e das notícias faz-se numa área que é adicionada abaixo desses elementos.
  • É dado um alerta ao utilizador se foi inserida uma nova notícia na base de dados
  • É utilizado AJAX e JQUERY extensivamente ao longo do site
  • O header do administrador possui um botão extra que permite aceder à zona de configurações do site, onde é possível adicionar ou remover servers remotos e as tags disponíveis na base de dados.
  • O administrador pode também fazer pesquisas noutros servidores e escolher qualquer uma das notícias para adicionar à base de dados. Pode também actualizar a notícia da base de dados caso já a tenha e os dados tenham sido actualizados.
